Inhoud

Ik heb deze POC gemaakt, omdat deze POC me heel leerzaam leek. Het is qua HTML, CSS en JavaScript

niet super veel werk. In de HTML heb ik 2 DIVS staan voor de "Speler" en de "Computer".

ik heb met JavaScript een aantal functies gemaakt die bijvoorbeeld de score bijhouden en de winnaar uitkiezen.

Feedback

De feedback die ik gekregen heb is dat het een prima POC is om mee te oefenen, vooral omdat

het qua JavaScript code nog best wel simpel is.

Wat heb ik ervan geleerd?

Bij deze POC heb ik vooral meer geleerd over JavaScript. Het is eigenlijk bijna hetzelfde

als C# of C++, alleen bij JavaScript hoeven sommige dingen niet zoals een punt-komma.

Ook zijn sommige termen wel wat anders, zoals een "var" of "let".
