Inhoud

We moesten in de Special Week aan de slag met een JavaScript framework. Ik heb toen gekozen voor

React, omdat hier veel over te vinden was. Verder heb ik naar vacatures gekeken en hier werd ook

vaak React op vermeld. Ik ben eerst begonnen met het onderzoeken wat React is, daarna ben ik

tutorials gaan kijken. Toen ik de basis een beetje snapte ben ik meteen aan de gang gegaan met een POC.

Feedback

De feedback die ik gekregen heb is dat het best goed is gelukt om iets te maken, omdat ik hier ook

niet eerder mee heb gewerkt. Daarnaast was de feedback dat het goed was dat ik eerst goed onderzoek

heb gedaan, voordat ik was begonnen met het maken van een POC.

Wat heb ik ervan geleerd?

Ik heb er erg veel van geleerd. Ik wist totaal niet wat React was of waar het voor gebruikt werd.

Na het onderzoeken van React was me al vrij duidelijk waar React voor gebruikt werd en hoe je het

zelf kon gebruiken. Daarnaast waren de tutorials erg duidelijk en behulpzaam.

Ik heb ook geleerd hoe je vanuit Node een React App kan maken en kan opstarten.

Ik heb geleerd hoedat je een React app kunt programmeren met Visual Studio Code.
