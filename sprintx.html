Inhoud

In Spint X ben ik aan de gang gegaan met het maken van een app in Android Studio. Sam was mijn begleider in Sprint X.

Ik wilde eerst een app maken voor merchandise, maar dat lukte me niet goed. Toen ben ik een app gaan maken waar je eten

op kunt bestellen. Dit lukte me wel goed. Ik had voor het maken van deze app eerst onderzoek gedaan naar Android studio en

ik had een prototype gemaakt in XD. Je kunt de app zelf niet runnen, want hier heb je de software van Android studio voor nodig.

In de video van Sprint X kijk ik ook naar de app.

Feedback

De eerste feedback die ik had gekregen was dat ik met usertesting aan de gang moest gaan. Dit heb ik toen in mijn planning

verwerkt. Ik heb met Sam nog een keer gebeld om het eindproduct te laten zien, hier was Sam tevreden over, hij vond dat

ik veel had gedaan en dat ik goed bezig was geweest.

Wat heb ik ervan geleerd?

Ik heb van deze opdracht veel geleerd. Ik heb vooral veel over Android Studio geleerd. Ik wist bijna niks van Android

Studio af. Ik heb geleerd hoe je met Android Studio moet werken, hoe je er code in kan schrijven en hoe je er een app

mee kunt designen. Verder heb ik geleerd hoe je je app op een Android telefoon kunt runnen. Daarnaast heb ik veel geleerd

over het designen van een prototype voor een App in XD.
