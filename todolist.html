Inhoud

Ik heb een To-Do-List gemaakt met JavaScript, HTML en CSS. Ik vond dit zelf een interessante opdracht,

omdat het niet echt een groot stuk code is, dus ik ben gaan kijken hoe ik de code zo klein en zo

makkelijk mogelijk kon maken. Verder kun je nog geen "taken" van de lijst verwijderen, daar heb ik nog

geen tijd voor gehad.

Feedback

De feedback die ik kreeg is dat het makkelijk en duidelijk om te gebruiken is. Het is jammer dat

er geen functie is om taken uit het lijstje te verwijderen, dat zou het een betere To-Do-List

maken.

Wat heb ik ervan geleerd?

Ik heb veel geleerd. Ik heb zoals bij elk project met JavaScript meer kennis en ervaring op gedaan.

Bij deze opdracht heb ik vooral geleerd hoe ik een programma kan maken met een klein stuk code.
